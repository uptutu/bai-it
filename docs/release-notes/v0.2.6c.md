# 掰it v0.2.6c Release Notes

**发布日期**: 2026-03-17

## 新功能

### 🎯 悬浮图标划词翻译
- 选中文本后，在选区右上角自动显示插件图标
- 点击图标即可快速翻译（与右键菜单功能相同）
- 支持英文单词、短语、句子翻译
- 淡入缩放动画效果，暗色模式自适应

### 📚 生词本管理页面
- Options 页面新增「生词本」Tab
- 列表展示：单词、音标、词性、释义、遭遇次数
- 按添加时间倒序排列
- 支持搜索单词或释义
- 单个删除和批量删除功能
- 空状态友好提示

### 💾 导出/导入功能
- **导出**: JSON 格式，包含元数据（版本、时间、单词数）
- 文件名格式：`bait-vocabulary-YYYY-MM-DD.json`
- **导入**: 合并模式，自动跳过已存在的单词
- 导入后显示统计结果（新增数、跳过数）

### ✨ 生词本单词高亮优化
- 生词本单词在页面上以**黄色高亮**显示
- 普通生词以蓝色虚线显示，视觉区分明确
- 鼠标悬停显示完整信息：单词、音标、词性、释义
- 添加/移除生词本后**实时更新**页面高亮

## 改进

- Firefox 浏览器支持
- 自定义 LLM Provider 配置
- 翻译结果悬浮框持久显示，添加关闭按钮
- 暗色模式/亮色模式/自动切换主题
- Anthropic API 响应解析兼容 thinking 块
